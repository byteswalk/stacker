3设为默认版本 · 弹窗弹窗GOROOT+PATH 捆绑同级写入；作用范围 + 显眼提权
Stacker

把默认 Go 切到 1.21.10
GOROOT 和 PATH 始终一起修改、同级同步。
仅当前用户 推荐免管理员
写 HKCU：用户级 GOROOT + 用户 PATH。命令 go 当前即经用户 PATH 解析，无需 UAC 提权。
系统全局需管理员 · 需 UAC 提权
写 HKLM：所有用户生效。仅当需要全局生效、或命令被系统 PATH 覆盖时选用。
改动前自动备份 GOROOT 与 PATH，可在「历史」还原。

下载 Go 版本
Go 在 Windows 上没有公认好用的版本管理器（gvm / goenv 偏 Unix），因此由 Stacker 直接下载官方 zip 解压、并切换 GOROOT。
不会装进 Stacker 程序目录；可自选磁盘。
