把默认 JDK 切到 17.0.10 · Temurin
JAVA_HOME 和 PATH 始终一起修改、同级同步——你只需选择作用范围，不会出现一个已改、一个未改的不一致情况。
仅当前用户免管理员
写 HKCU：用户级 JAVA_HOME + 用户 PATH（一起）。无需 UAC 提权。
系统全局 推荐需管理员 · 需 UAC 提权
写 HKLM：系统级 JAVA_HOME + 系统 PATH（一起），对所有用户生效。
检测到覆盖：系统 PATH 中的 javapath 会覆盖命令。若仅写用户级，终端 java 仍为旧版本——因此默认选择系统级。
改动前自动备份当前 JAVA_HOME 与 PATH（用户级 + 系统级），可在「历史」一键还原。
