4设为默认版本 · 弹窗弹窗MAVEN_HOME+PATH 捆绑同级写；Maven 通常用户级即够，仍给系统级
Stacker
Maven
把默认 Maven 切到 3.8.8
MAVEN_HOME 和 PATH 一起改、同级同步。
仅当前用户 推荐免管理员
写 HKCU：用户级 MAVEN_HOME + 用户 PATH。命令 mvn 当前即经用户 PATH 解析，已满足需求，无需 UAC 提权。
系统全局需管理员 · 需 UAC 提权
写 HKLM：所有用户生效。仅当需要全局生效、或命令被系统 PATH 覆盖时选用。
改动前自动备份 MAVEN_HOME 与 PATH，可在「历史」还原。
